Frequently asked Questions

Here are some of the frequently asked questions

What is roadmap.sh?
A community effort to create roadmaps, guides and other educational content to help guide developers in picking up a path and guide their learnings.
What are the plans for Roadmap.sh?
Plans to expand beyond just visual guides by adding features like best practices, quizzes, project ideas, public profiles to track progress, and interactive elements to enhance the learning experience
How is Roadmap.sh Built?
The website is built with Astro and Tailwind and is deployed to GitHub pages. The project is open-core and the codebase can be found on GitHub
Can I use Roadmap.sh in my team?
Yes, you can use Roadmap.sh with your team; it's designed to provide structured learning paths and career development roadmaps for developers at various levels, allowing you to create custom team roadmaps, track individual progress, and identify skill gaps within your team, all through a community-driven platform.
How can I create custom Roadmaps?
Start by clearly defining your project goals and key milestones, then break down the work into phases, assigning timelines and responsibilities to each task while utilizing a visual tool like a project management software or design platform to represent the plan with customized elements like colors, icons, and swimlanes
